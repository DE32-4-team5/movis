# 🎬 영화 조회 웹 애플리케이션

이 애플리케이션은 Java Spring Framework와 영화진흥위원회 영화 목록 API를 사용하여 영화 정보를 조회할 수 있는 웹 애플리케이션입니다.

## 목차

- [기술 스택](#기술-스택)
- [사용 방법](#사용-방법)

## 소개

이 애플리케이션은 사용자가 영화진흥위원회 API를 통해 최신 영화 목록을 조회하고, 각 영화의 상세 정보를 확인할 수 있도록 합니다. 사용자 친화적인 인터페이스를 제공하여 쉽게 사용할 수 있습니다.
